# LoadSprint Driver — приложение для iPhone и iPad (Expo / React Native)

Это водительское приложение для App Store. Оно подключается к твоему уже
работающему сайту (Railway/Vercel) и использует те же грузы, документы, фото,
чат, статусы и уведомления.

---

## ВАЖНО: сначала впиши адрес своего сайта

Открой файл **`src/config.ts`** и замени одну строку:

```ts
export const API_BASE = "https://YOUR-SITE-URL-HERE";
```

Поставь сюда адрес своего сайта на Railway или Vercel, без слэша в конце.
Например: `https://loadsprint-production.up.railway.app`

Это единственное обязательное изменение. Без него приложение не найдёт сервер.

---

## Что нужно установить на компьютере (Mac НЕ нужен)

1. **Node.js 18+** — https://nodejs.org (бери LTS).
2. Аккаунт **Expo** (бесплатный) — https://expo.dev (зарегистрируйся).
3. Для публикации в App Store — аккаунт **Apple Developer** ($99/год) —
   https://developer.apple.com . Без него можно тестировать, но не выложить.

Mac не нужен: Expo собирает iOS-сборку в облаке (EAS Build).

---

## Запуск и проверка на своём телефоне (5 минут)

В терминале, в папке `loadsprint-driver`:

```bash
npm install
npx expo start
```

На айфоне установи приложение **Expo Go** из App Store, открой камеру и наведи
на QR-код из терминала. Приложение откроется прямо на телефоне — можно проверить
вход, грузы, навигацию, фото, чат.

> Камера и съёмка фото в Expo Go работают. Глубокие ссылки на Trucker Path
> полноценно работают только в настоящей сборке (шаг ниже), не в Expo Go.

---

## Сборка для App Store (без Mac, через облако EAS)

Один раз установи инструмент и войди:

```bash
npm install -g eas-cli
eas login                 # вход в аккаунт Expo
eas init                  # создаст projectId и впишет его в app.json
```

Собери iOS-версию (EAS попросит данные Apple Developer и сам создаст сертификаты):

```bash
eas build --platform ios --profile production
```

Сборка идёт в облаке ~10–20 минут. По готовности получишь файл `.ipa`.

Отправка в App Store Connect:

```bash
eas submit --platform ios --latest
```

Дальше — в https://appstoreconnect.apple.com: заполняешь описание, скриншоты,
политику конфиденциальности и отправляешь на ревью Apple.

## Про навигатор

Кнопки навигации открывают системную карту (Apple Maps / Google Maps) и
**Trucker Path** (truck-legal маршруты для фур), если он установлен на телефоне.
Настоящая встроенная трак-маршрутизация (мосты, вес, опасные грузы) требует
платного API (Mapbox/HERE/Google) — это отдельный платный шаг, который можно
добавить позже.
